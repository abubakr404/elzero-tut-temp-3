## Run Locally

Clone the project

```bash
  git clone https://github.com/abubakr404/elzero-tut-temp-3.git
```

Go to the project directory

```bash
  cd template_3
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
